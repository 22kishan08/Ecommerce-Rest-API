# Ecommerce-Rest-API

### Installation 

#### Download project

```bash
git clone https://github.com/22kishan08/Ecommerce-Rest-API
```

#### Install Dependancies 
Run following command: 
```bash
cd Ecommerce-Rest-API
```
```bash
npm install
```

#### Setup Environment 
```bash
cp .env.example .env
```
Change DB_URL to yours db url. 

#### Run the server
```bash
npm run dev
```
